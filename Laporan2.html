BAB 4. LAPORAN TEKNIKAL

4.3-Barcode Scanner

Tugasan yang seterusnya adalah setup Barcode Scanner iaitu BHT (Barcode Handy Terminal) yang digunapakai dalam syarikat ini. Pengimbas 'Barcode Scanner' ini berfungsi untuk mengambil gambar dalam kode bar, lalu dengan teknik pengolahan citra digital, kode dapat dibaca dan telihat deskripsi produk dan keterangan selengkapnya. Jenis mesin yang digunakan adalah 'Portable Data Terminal (PDT)' yang berkait rapat dengan 'Hand-held Terminal BHT'. Jenis ini menggunakan bateri yang boleh dicas semula pada 'Communciation Cradle' serta menyimpan data ke memori dan kemudian dapat diupload, dan data tersebut dapat kembali diunggah.

Dalam syarikat ini, Barcode Scanner iaitu BHT (Barcode Handy Terminal) kerap kali digunakan pada bahagian 'Store', 'Assembling' dan 'Slitting' untuk kerja merekod data, menghantar data dan banyak fungsi yang lain untuk memudahkan syarikat dalam menghasilkan kapasitor dengan lebih cepat dan tepat. Ini memainkan peranan penting untuk IT support melalukan penyelenggaran mesin tersebut sekiranya ada ralat sistem yang berlaku dan membuat setup BHT yang baharu. BHT juga hanya boleh disambung ke rangkaian melalui sambungan Wi-Fi mengikut alamat IP yang ditetapkan untuk setiap unit BHT. Ini bertujuan untuk dihubungkan ke 'Server' seperti memberikan sambungan ke 'Thermal Printer' untuk melakukan 'Print Sleeve' mengikut data yang telah diambil oleh BHT.



Semasa menjalani latihan industri, pelajar dikehendaki membuat setup BHT (Barcode Handy Terminal) yang baharu seperti menetapkan alamat IP. Dan juga membuat penyelarasan tetapan pada 'System Menu' seperti tetapkan masa dan tarikh, upload, download dan delete fail yang menyebabkan ralat dalam sistem jika berlaku. Antara langkah untuk membuat setup BHT (Barcode Handy Terminal) adalah seperti berikut :-

i. Langkah Pertama, sambungkan BHT ke komputer dengan memakai kabel 'USB Type C' dan kemudian upload file dari komputer ke BHT selama beberapa minit. Untuk ke 'System Menu' adalah seperti berikut :-

i. Tekan [Pwr]+[SF]+[1]
Additional command :
[ENT] untuk memilih atau mengesahkan
[C] untuk membatalkan atau kembali
[SF] + [BS] untuk menukar abjad kepada nombor atau kembali ke huruf.

ii. Langkah Kedua, setelah 'Upload File', tentukan masa dan tarikh untuk BHT.



i. Dari SYSTEM MENU tekan 4: SET SYSTEM dan kemudian pergi ke 3: DATE/TIME.
ii. Masukkan tarikh (DATE) menggunakan format YEAR/MONTH/DAY
iii. Seterusnya masukkan masa (TIME) menggunakan format HOUR : MINUTE

iii. Langkah Ketiga, tetapkan alamat IP yang berlainan untuk setiap BHT.



i. Dari SYSTEM MENU tekan 4: SET SYSTEM dan kemudian pergi ke 9: TCP/IP
ii. Seterusnya 1: SET TCP/IP dan pergi ke 2: IP ADDRESS.

iv. Langkah Keempat, tetapkan FTP (Server)



i. Dari SYSTEM MENU tekan 4: SET SYSTEM dan kemudian pergi ke 9: TCP/IP
ii. Seterusnya 2: SET FTP dan pergi ke 1: SERVER

v. Langkah Kelima, Install Wi-Fi dalam BHT.



i. Dari SYSTEM MENU tekan 8: RF dan kemudian pergi ke 1: PARAMETER seterusnya 1: RF NETWORK
ii. Seterusnya pergi ke 1: NETWORK PARAMETER dan massukan Wi-Fi ke dalam SSID.

vi. Langkah Keenam, menguji sambungan Wi-Fi.



i. Untuk menguji sambungan Wi-Fi, Dari SYSTEM MENU tekan 8: RF dan kemudian pergi ke 2: SITE SURVEY
ii. Seterusnya pergi ke 1: LINK STATUS. BHT akan memaparkan 'Signal Strength Wi-Fi'.

vii. Langkah Ketujuh, 'Install Program' melalui 'Download File' dan kemudian 'Execute Program' melalui 'Set System'.



i. Dari SYSTEM MENU tekan 7: FTP dan kemudian pergi ke 1: DOWNLOAD.
ii. Seterusnya, masukkan nama file program dan download file tersebut.

viii. Langkah Kelapan, Execute Program melalui set system.



i. Dari SYSTEM MENU tekan 1: EXECUTE PROGRAM dan pilih program yang hendak digunakan.

ix. Langkah Sembilan, Hidupkan semula untuk menguji program tersebut dan sekiranya berjaya masuk ke dalam program tersebut menunjukkan BHT sudah boleh digunakan.



x. Langkah Kesepuluh, sekiranya ralat berlaku seperti gambar rajah di bawah. IT support perlu membuang ‚ÄúError File‚Äù dengan memasuki System Menu untuk melakukan delete file. Untuk membuang file tersebut adalah seperti berikut.



i. Dari SYSTEM MENU tekan SF+0.
ii. Kemudian delete file yang menyebabkan ralat berlaku.
